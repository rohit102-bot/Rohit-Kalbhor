## 💫 About Me

I’m **Rohit Kalbhor**, a dedicated **Cloud Engineer** with a deep focus on designing secure, scalable, and highly available infrastructure. I bridge the gap between core networking and modern cloud-native technologies.

- ☁️ I specialize in **AWS Cloud Architecture**, leveraging managed services to optimize performance and cost.
- 🐧 I am an expert in **Linux System Administration** (RHEL), with a strong emphasis on automation and server hardening.
- 🌐 My background in **Cisco Networking** (CCNA) allows me to design complex VPC structures and hybrid-cloud connectivity.
